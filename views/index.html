API Project: URL Shortener Microservice

User Stories:

  1. 1. I can POST a URL to [project_url]/api/shorturl/new and I will receive a shortened URL in the JSON response.
  2. Example : {"original_url":"www.google.com","short_url":1}
  3. If I pass an invalid URL that doesn't follow the http(s)://www.example.com(/more/routes) format, the JSON response will contain an error like {"error":"invalid URL"} HINT: to be sure that the submitted url points to a valid site you can use the function dns.lookup(host, cb) from the dns core module.
  4. 3.When I visit the shortened URL, it will redirect me to my original link.
  5. Short URL Creation
  6. example: POST [project_url]/api/shorturl/new - https://www.google.com

Example Usage:

  • [this_project_url]/api/shorturl/3
